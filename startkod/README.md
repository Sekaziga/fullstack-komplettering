# Startkod

Minimal fullstack-app (API i `api/`, frontend i `frontend/`) att utgå från. Docker, tester och health check ska du lägga till själv.

**Instruktioner:** Se **README** i projektets rot för hela kompletteringsuppgiften och steg-för-steg.

**Kör lokalt (utan Docker):** `cd api && npm install && npm start` i en terminal, `cd frontend && npm install && npm run dev` i en annan. Frontend: http://localhost:5173, API: http://localhost:3000.
